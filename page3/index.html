Do Something... }; </code></pre> <p>This isn't exactly ideal, what we would prefer to have is a more semantic route.</p> <ul> <li><code>GET</code> -> gets an object</li> <li><code>DELETE</code> -> deletes an object</li> <li><code>PUT</code> -> modifies an object</li> <li><code>PATCH</code> -> modifies part of an object</li> <li><code>POST</code> -> creates an object</li> </ul> <p><span class="note"><strong>Note:</strong> I'm no expert on this stuff so the VERBs above may not be correct wording.</span></p> <p>This would allow us to define the same route like so</p> <pre><code>Delete["/products/{id}"] = _ =&gt; { ...
